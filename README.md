# CigTable

The library is built for displaying 2D table matrix as a table on UI on the fly. The library is very basic and development is still is in progress.

### Usage

```html
<cig-table 
    [data]=[
        ["id","name","address"],
        [1503,"John Doe","San Jose"]
    ]>
</cig-table>
```
